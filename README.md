OneGuy Studio Simulation

A sophisticated, interactive 2D simulation of a developer's workspace, built with HTML5, Tailwind CSS, and GSAP for smooth animations.

Features

Interactive Lighting: Drag the pull cord to toggle between "Work Mode" and "Sleep Mode".

Dynamic Animations: Realistic character movements, coffee sipping routine, eye blinking, and cat breathing.

Flashlight Effect: Explore the dark room with a dynamic flashlight effect controlled by your mouse.

Responsive Design: Optimized for desktop and includes a rotation prompt for mobile users.

Social Links: Integrated Discord and GitHub links with hover effects.

Technologies Used

HTML5 / SVG: For structure and vector graphics.

Tailwind CSS: For rapid and responsive styling.

GSAP (GreenSock Animation Platform):

TweenMax: Core animation engine.

Draggable: For the interactive light pull cord.

TextPlugin: For text effects (if enabled).

How to Use

Clone this repository.

Open index.html in any modern web browser.

Interact:

Move your mouse to use the flashlight in the dark.

Drag the red knob on the rope downwards to switch the lights on/off.

Hover over the wall frames to see interactions.
